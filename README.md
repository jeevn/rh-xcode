## Installation

Clone repo:

    $ git clone https://github.com/jeevn/rh-xcode

Move theme files:

    $ cd rh-xcode/themes
    $ mv *.dvcolortheme ~/Library/Developer/Xcode/UserData/FontAndColorThemes/

Now relaunch Xcode, the themes should appear in Preferences.
